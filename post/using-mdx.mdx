此主题已在 `astro.config.mjs` 配置文件中安装并配置了 [@astrojs/mdx](https://docs.astro.build/en/guides/integrations-guide/mdx/) 集成。如果您不想使用 MDX，可以通过从配置文件中移除该集成来禁用支持。

## 为什么使用 MDX？

MDX 是一种特殊的 Markdown 格式，支持嵌入 JavaScript 和 JSX 语法。这使得您可以 [将 JavaScript 和 UI 组件混合到 Markdown 内容中](https://docs.astro.build/en/guides/markdown-content/#mdx-features)，例如交互式图表或提示框。

如果您已有使用 MDX 编写的内容，此集成应该能让您轻松迁移到 Astro。
